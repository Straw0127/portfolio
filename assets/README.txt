# Assets Folder

## Profile Image
Place your profile photo here:
  assets/profile.jpg

## Project Screenshots
Place screenshots inside each project's folder:

  assets/techplay/       → for project-techplay.html
  assets/attendance/     → for project-attendance.html
  assets/pixelhub/       → for project-pixelhub.html
  assets/ecommerce/      → for project-ecommerce.html
  assets/crm/            → for project-crm.html

## Supported Image Formats
  - .jpg / .jpeg
  - .png
  - .webp

## Naming Convention
You can name your screenshots any of these ways:
  1.jpg, 2.jpg, 3.jpg ...         (numbered)
  screenshot1.jpg, screenshot2.jpg (prefixed)

The gallery will auto-detect and display them in order.

## Optional: manifest.json
For explicit control, create a manifest.json inside a project folder:

  {
    "images": ["dashboard.png", "login.png", "quiz.png"]
  }

This loads images in the exact order specified.
