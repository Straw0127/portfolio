Python Desktop Application

Student Attendance
System

A Python-based desktop application built to replace manual, paper-based attendance tracking in classrooms. The system provides a clean GUI for recording and reviewing student attendance, managing student records, and tracking sessions.

Built with Python and tkinter, this project focused on creating something practically useful — the kind of tool that actually saves time in a real academic environment.

Features

What the system includes

📋
Attendance Recording

Mark students as present, absent, or late with a simple click interface. Session-based logging for each class day.

👨‍🎓
Student Records

Add, edit, and manage student profiles. View individual attendance history and overall attendance rates.

📅
Session Management

Track attendance per session or date. Review previous sessions and make corrections when needed.

📊
Attendance Reports

Generate summaries showing each student's attendance record — useful for grading and teacher reports.

🖥️
Desktop GUI

Built with Python's tkinter for a clean, familiar desktop interface. No browser needed, no setup required beyond Python.

💾
Data Persistence

Attendance data saved locally so records are available between sessions without any server or internet connection.
