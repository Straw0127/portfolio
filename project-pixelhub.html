Web Gaming Platform

PixelHub
22+ Web Games

PixelHub is a browser-based gaming platform featuring 22+ games built entirely with vanilla HTML, CSS, and JavaScript. No game engines, no external libraries — each game is hand-coded from scratch.

This project was a deep dive into JavaScript game logic, canvas rendering, collision detection, and browser-based interactivity. Building 22+ different games forced creativity and problem-solving at every step.

Features

What PixelHub includes

🕹️
22+ Playable Games

A wide variety of browser games — from classic arcade-style games to puzzles and action games, all playable in the browser.

🎨
Vanilla JS Only

Zero external libraries or frameworks. Every game is built with pure HTML, CSS, and JavaScript — good for learning and performance.

🖼️
Canvas Rendering

Many games use HTML5 Canvas API for sprite rendering, animations, and real-time game loop logic.

🏠
Game Hub / Launcher

A central platform UI for browsing and launching all available games — organized and accessible.

📱
Browser-Based

No installation required. Open in any modern browser and play instantly — works on desktop and most touch devices.

🧠
Game Logic Variety

Covers collision detection, scoring systems, timers, difficulty scaling, and different input methods across 22+ titles.
