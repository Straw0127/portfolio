Features

What the platform includes

🛍️
Product Catalog

Browse products with images, descriptions, and pricing. Category filtering and search functionality included.

🛒
Shopping Cart

Session-based cart system — add, remove, and update item quantities. Cart persists across page navigation.

💳
Checkout Flow

Order form with customer details, order summary, and confirmation page. Clean multi-step process.

⚙️
Admin Panel

Manage products (add/edit/delete), view orders, and update product inventory from a simple admin interface.

🔐
User Authentication

Registration and login for customers. Session management and basic access control between user and admin roles.

🗄️
MySQL Backend

Relational database storing products, customers, orders, and order items with proper table relationships.
