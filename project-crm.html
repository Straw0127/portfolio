CRM System

A Customer Relationship Management application built as a Java desktop program using Swing in the NetBeans IDE. This project was developed as part of coursework to demonstrate OOP design principles and GUI application development.

While Java isn't my primary language, this project shows my understanding of object-oriented programming, class design, and desktop GUI construction.

Features

What the CRM includes

👤
Contact Management

Add, edit, view, and delete customer records with full details — name, contact info, notes, and status.

🔍
Search & Filter

Quickly find customer records using real-time search and filter controls within the data table.

📋
Data Tables

JTable-based dynamic data display with sortable columns and clean record presentation.

🖥️
Swing GUI

Built with Java Swing — custom forms, panels, dialogs, and buttons designed for usability in NetBeans.

🏗️
OOP Architecture

Clean object-oriented design with separate classes for UI, data models, and business logic — as required for coursework.

💾
Data Persistence

Customer records saved and loaded reliably, maintaining data between sessions without a server.
