About Me

Crafting Systems.
Building Experiences.

Hi, I'm Justine Serdinola — an IT/CS student who actually builds things. Not just tutorial projects, but real systems: a gamified LMS, a Python attendance tracker, a 22-game web arcade, and more.

My main stack is PHP & MySQL for backend systems and web applications, Python for scripts and desktop tools, and HTML/CSS/JS for the frontend. Java is part of my coursework — I've built a CRM with it, but it's not my go-to.

I'm the kind of student who sees a problem in a school system and thinks "I could build something that fixes that." TechPlay came from wanting to make learning less boring. The Attendance System came from seeing how manual processes waste everyone's time.

Still studying, still building. If you're a fellow IT student who needs help fixing a project, debugging code, or building something from scratch — I'm open to working together on a paid or freelance basis.
